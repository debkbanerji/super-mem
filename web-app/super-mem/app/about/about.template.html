About Memento

Memento is a platform to demonstrate the power of an extremely efficient compression algorithm for storing composite captioned images. Here you can view compressed images, or download a very lightweight implementation of a .meme file interpreter written in JavaScript

Created by Alex Hirschberg, Deb Banerji and Ragavi Ramesh

Getting Started

In order to use the interpreter, first download either the regular or minified JavaScript file and reference it on your page.

Download JavaScript Download Minified JavaScript

After you've included the JavaScript file, simply call drawMeme(canvas, meme) in order to unpack the image and draw it on your page, where canvas is the HTML5 canvas element on which the image will be rendered and meme is the object corresponding to the .meme file containing the image.
