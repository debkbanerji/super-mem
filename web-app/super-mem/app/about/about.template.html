About .meme

.meme is a platform to demonstrate the power of an extremely efficient compression algorithm for storing composite captioned images. Here you can view compressed images, or download a very lightweight implementation of a .meme file interpreter written in Javascript

Created by Alex Hirschberg, Deb Banerji and Ragavi Ramesh

Getting Started

In order to use the interpreter, first, download either the regular or minified Javascript file and reference it on your page.
